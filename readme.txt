git 命令大全

1.git init
2.git add 
3.git commit -m '说明'
4.git remote add origin https://github.com/Ranting1992/gitLearn.git  //本地仓库与远程仓库关联
5.git push -u origin master   //推送，u参数会把本地的master分支内容推送的远程新的master分支，
							  //还会把本地的master分支和远程的master分支关联起来，在以后的推送或者拉取时就可以简化命令。
